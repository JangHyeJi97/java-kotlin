무궁화 삼천리 화려강산 대한사람 대한으로 길이 보전하세

공뱁 문자의 처리를 위해 white-space 속성을 사용할 수 있음

자동 줄바꿈을 허용하지 않음 경우 white-space:nowrap; 형식으로 선언하면 한 줄에 출력할 수 있다. 만일 웹 브라우저 크기를 넘어갈 만큼의 넓이 라면 스크롤바가 나타날 수도 있다.

하나 이상의 연속적인 공백이 있을 때 입력한 형태대로 출력하려고 한다면 white-space:pre; 형식으로 선언하면 된다.
